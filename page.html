Красава, что зашел!

MegapsOS — это операционка, которую мы пишем на C# под Cosmos. Никакого выпендрежа, просто рабочий инструмент.

Файловая система FAT32, 20+ команд, цветной интерфейс и звуки как на Sega. Всё работает из коробки.

Чё за зверь?

MegapsOS — это операционка для тех, кто шарит. Не очередной Linux на коленке, а своя архитектура от и до.

По факту: Легкая, быстрая, делает что скажешь. Загружается за секунды, жрет минимум памяти, не собирает телеметрию и не просит обновляться по 100 раз.

Для кого это?

Для разработчиков, энтузиастов и просто пацанов, которым интересно как там под капотом. Если ты лох — ставь Windows, а если пацан — оставайся с нами.

Ваще: Версия 0.6.8 уже вполне рабочая. Можно файлы создавать, удалять, по папкам ходить, системой управлять. И звуки есть — как на Sega.

Документация

Быстрый старт

Скачал ISO — загружайся на виртуалку или реальный комп. Всё само встанет.

MegapsOS:0:\> help          # помощь
MegapsOS:0:\> ver           # версия
MegapsOS:0:\> raminfo       # сколько ОЗУ
MegapsOS:0:\> ls            # список файлов
MegapsOS:0:\> cd games      # зайти в папку games
MegapsOS:0:\> mf readme.txt # создать файл

Навигация по файлам

Команда Описание Пример
cd Переход по папкам cd system или cd ..
ls / dir Показать содержимое ls
mkdir / md Создать папку md games
touch / mf Создать файл mf config.cfg
cat / type Посмотреть файл cat readme.txt
rm / rf Удалить файл rm trash.txt

Все команды

help Помощь по системе
ver Версия ОС
datetime Дата/время
raminfo Инфо по RAM
clear/cls Очистка экрана
shutdown Выключение
reboot Перезагрузка
echo Вывод текста
ls/dir Список файлов
cd Сменить папку
mkdir/md Создать папку
rmdir/rd Удалить папку
touch/mf Создать файл
rm/rf Удалить файл
cat/type Прочесть файл
system list Детальный список
run calc Калькулятор

Для ленивых есть system makefile, system makedir и другие — всё в интерактивном режиме с подтверждениями.
